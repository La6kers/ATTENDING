## BioMistral Configuration

The system supports multiple AI providers:

| Provider | Use Case | Setup |
|----------|----------|-------|
| Ollama | Local development | `ollama pull biomistral:7b` |
| HuggingFace | Cloud deployment | Set `BIOMISTRAL_API_KEY` |
| Replicate | Serverless | Set `REPLICATE_API_TOKEN` |
| OpenAI-compatible | Custom deployment | Set endpoint URL |

All providers automatically fall back to rule-based logic if unavailable.
